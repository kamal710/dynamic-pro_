# Discord.js Handler
Dynamic Pro, rewrite

## SelfHost
- Click on this image:
![Code](https://i.imgur.com/x1dUOfK.png)
- And "Download ZIP"
- Extract the file
- Open with your code editor
- Run `npm i` in terminal
- Create a `.env` file and put `TOKEN=YOUR_DISCORD_BOT_TOKEN` in it.
- run `npm start`
